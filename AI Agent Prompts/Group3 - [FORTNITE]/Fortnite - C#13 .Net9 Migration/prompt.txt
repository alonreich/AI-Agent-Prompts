## 2. NATIVE INTEROP & THREAD SAFETY
### 2.1 MPV P/Invoke & Deadlock Prevention
- Direct P/Invoke: Use direct `[LibraryImport]` source-generated bindings to `libmpv-2.dll`. Avoid heavy third-party wrappers.
- Unmanaged Delegate GC Pinning: Mandate explicit use of `GCHandle.Alloc` or `[UnmanagedCallersOnly]` for all C# callback delegates passed to unmanaged code. Unpinned delegates will trigger Native Access Violations (`0xC0000005`).

### 2.2 FFmpeg Hardware Orchestration & Subprocess Safety
- Subprocess Pipe Deadlocks: Implement asynchronous stream reading via `BeginOutputReadLine()` and `BeginErrorReadLine()` for all `ffmpeg.exe` and `ffprobe.exe` process instances. Standard synchronous reads will saturate the OS pipe buffers and freeze background worker threads.

- Priority Pipeline: Execution selection order is immutable: `NVIDIA (h264_nvenc)` -> `AMD (h264_amf)` -> `INTEL (h264_qsv)` -> `CPU (libx264)`.
